## Installation
Add this line to your application's Gemfile:

Gemfile
```ruby
gem 'fall_snow'
```

And then execute:
```bash
$ bundle
```

app/assets/stylesheets/application.css
```application.css
@import 'fall_snow/style.css';

body{
     background-color: #142744;
 }
```

view_file
```view_file
<%= fall_snow %>
```

<!-- Or install it yourself as:
```bash
$ gem install fall_snow
``` -->
