## Installation
Add this line to your application's Gemfile:

```ruby
gem 'fall_snow'
```

And then execute:
```bash
$ bundle
```

style.css
```style.css
@import 'fall_snow/style.css';
```

view_file
```view_file
<%= fall_snow %>
```
